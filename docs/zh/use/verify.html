# 如何验证数据

DartMars 默认提供了一些常用的验证数据的方法，你可以使用 VerifyHelper 来使用他们。

例如验证某字符串是不是空字符串

String s1 = '';
true == VerifyHelper.empty(s1);

String s2;
true == VerifyHelper.empty(s2);

int n1 = 0;
